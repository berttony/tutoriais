# tutoriais
Tutoriais organizados conforme minha mente que vou encontrando pelo caminho.

## Árvore de conteúdos

- src/
  - chrome/
    - [chrome-crash-solution.md](src/chrome/chrome-crash-solution.md) — Guia para identificar e resolver travamentos no Google Chrome: limpeza de cache, gerenciamento de extensões, desativar aceleração de hardware e reinstalação.
  - wsl2/
    - [wsl2-docker.md](src/wsl2/wsl2-docker.md) — Comandos e passos para instalar e configurar o Docker dentro de uma distribuição WSL2 (inclui comandos de instalação e configuração do Portainer).
    - [wsl2-mkdocs.md](src/wsl2/wsl2-mkdocs.md) — Como configurar MkDocs em WSL2, criar um Dockerfile quando necessário (para o plugin mermaid2) e subir containers para servir documentação.
    - [wsl2-ubuntu.md](src/wsl2/wsl2-ubuntu.md) — Passos iniciais para instalar e configurar WSL2 no Windows, comandos úteis do `wsl` e verificação de versão.

## Como usar

- Clique nos links para abrir os tutoriais individuais.
- Os caminhos são relativos ao repositório — se navegar no GitHub, os links abrem os arquivos `.md` correspondentes.

## Contribuições

Adicione novos tutoriais em `src/` e atualize esta árvore com um link e uma breve descrição.
